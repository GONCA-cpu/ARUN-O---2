 English Description

This project is an **Arduino-based pedestrian traffic light system** designed and simulated in **Tinkercad**.  
It includes red, yellow, and green LEDs for vehicles, and red/green LEDs for pedestrians.  
When a pedestrian presses the button, the system stops vehicle traffic and allows safe crossing.  
After a few seconds, it automatically returns to normal operation.
